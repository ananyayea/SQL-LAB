# SQL Lab – Experiment 1

## Aim
To create EMPLOYEE and DEPARTMENT tables with given constraints and perform DDL and DML operations.

---

## Table Creation

### EMPLOYEE Table

### Query
```sql
CREATE TABLE EMPLOYEE (
    EMPNO NUMBER(4) PRIMARY KEY,
    ENAME VARCHAR2(20) NOT NULL,
    JOB VARCHAR2(20),
    MGR NUMBER(4),
    HIREDATE DATE,
    SAL NUMBER(10),
    COMM NUMBER(7),
    DEPTNO NUMBER(2)
);
```

---

### DEPARTMENT Table

### Query
```sql
CREATE TABLE DEPARTMENT (
    DEPTNO NUMBER(2) PRIMARY KEY,
    DNAME VARCHAR2(15) NOT NULL
);
```

# Perform Following Queries

---

## 1. Create Employee_Master table with data using Employee table

### Query
```sql
CREATE TABLE EMPLOYEE_MASTER AS
SELECT * FROM EMPLOYEE;
```

---

## 2. Delete all records from Employee_Master whose DeptNo is 10

### Query
```sql
DELETE FROM EMPLOYEE_MASTER
WHERE DEPTNO = 10;
```

---

## 3. Update 10% increase in salary of DEPTNO 20 in Employee_Master

### Query
```sql
UPDATE EMPLOYEE_MASTER
SET SAL = SAL + (SAL * 0.10)
WHERE DEPTNO = 20;
```

---

## 4. Alter SAL with size 10,2 in Employee_Master

### Query
```sql
ALTER TABLE EMPLOYEE_MASTER
MODIFY SAL NUMBER(10,2);
```

---

## 5. Drop Employee_Master Table

### Query
```sql
DROP TABLE EMPLOYEE_MASTER;
```
